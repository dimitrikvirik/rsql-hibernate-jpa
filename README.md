# rsql-hibernate-jpa

RSQL implementation for Hibernate/Spring data with join tables support

# Dependencies:
* Hibernate Core 5.4.21.Final
* Spring Data JPA 2.2.9.RELEASE
* RSQL Parser - [rsql-parser](https://github.com/jirutka/rsql-parser)
